# TME Starter Add-On BOM

This is a separate through-hole starter BOM for Eurorack-style DIY work. It is meant to complement the main TME order and stay focused on reusable CV/gate and analog audio building blocks.

The target budget is roughly 200 PLN at TME, depending on current stock and package choices.

## Suggested Quick-Build Targets

- Buffered CV input and gate conditioner
- Simple active mixer / attenuverter
- Two-pole low-pass or band-pass filter
- Envelope follower / comparator gate extractor
- LFO or clock source
- Basic utility VCA or OTA filter experiment with LM13700N

## Notes

- Everything below is through-hole.
- These are middle-ground parts, not premium audio-only selections.
- Analog delay and chorus are intentionally left for a later, more specialized order.
