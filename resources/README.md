# Scene Resources

Place per-scene assets here.

Suggested layout:
- `fonts/` font files used by shader-generated text atlases
- `videos/` playback sources
- `data/` arbitrary scene data files
- `textures/` bitmap texture resources

The MIDI dot scene expects a note-label font file such as `fonts/fa-solid-900.otf` when you want the atlas to use a scene-provided font.
